# gift__tracker - a simple web app to practice various ideas
# gt allows a user to track recipients of Christmas gifts
# User can add new lists, representing a person, add name, and list of gifts ideas for the person. 
# As a gift is purchased or decided against, the item can be removed
# If a person has been naughty, their list can be deleted completely.
# Uses the Local Storage object to persist the lists, and arrays in memory to manage the data.
